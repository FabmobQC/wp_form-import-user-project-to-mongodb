# Description

Tool to import Wordpress WP Form field and insert it in the MongoDB

# Usage

Create an .env file following this example

```
{
        "databases": {
                "mongodb": {

                        "url": MONGODB_CONNECTION_STRING
                },
                "mysql": {
                        "host": HOST,
                        "user": USERNAME,
                        "password" : PASSWORD,
                        "dbname": DATABASE_NAME
                }
        }
}
```
